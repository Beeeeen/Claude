Always use WebSearch to get the latest information before answering questions about current products, prices, market conditions, software versions, or any time-sensitive topics. Do not rely solely on training knowledge.

**Why:** Training data has a cutoff and can be outdated. User was told M5 hadn't released when it already had — this kind of stale answer is not acceptable.

**How to apply:** Whenever the question involves anything that could have changed (hardware specs, pricing, releases, regulations, market data), search first, then answer.
